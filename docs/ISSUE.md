## 관리자 코멘트

- 라이프보드의 메인 분류는 인간의 필수 욕구에서부터 가지처럼 상세분야로 뻗어나가는 틀로 만들면 좋을거 같음.(메슬로의 욕구단계 이론 참고)

## 사용자 피드백

### 1. 뉴스

- 대시보드나 뉴스 페이지 접속 시 보여지는 뉴스는 네이버뉴스 처럼 대표이미지와 제목만 표시되게 하면 좋을거 같음.
- 간단하게 표시된 뉴스를 보고 내용을 보고싶으면 눌러서 상세 뉴스 페이지로 들어가게 해줘.
- 뉴스마다 정리된 폼이 달라서 눈에 잘 안들어오니 하나의 폼을 만들어서 정리된 내용을 표시하게 해줘.
- AI가 요약된 내용을 보니 중국어로 되어 있는거도 있어서 한국어로만 요약하게 해줘.
- 대부분의 정리된 뉴스마다 관련기사가 하나씩 밖에 없어서 정리 시스템이 제대로 동작되지 않는거 같음.
- 연예인 일상적인 내용이나 유명인 경조사 등 우리 생활에 필요없는 단순한 내용의 기사는 안보고 싶음.
- AI가 요약을 못하거나 이상하게 한 기사는 안보였으면 좋겠음.
