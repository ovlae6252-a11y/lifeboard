## 관리자 코멘트

- 라이프보드의 메인 분류는 인간의 필수 욕구에서부터 가지처럼 상세분야로 뻗어나가는 틀로 만들면 좋을거 같음.(메슬로의 욕구단계 이론 참고)
- 현재 올라마의 주요내용 작성이 너무 마음에 안듦. 사용자의 입장에서는 정리된 뉴스를 보기를 원하는데 실제로는 그냥 한줄 요약 느낌으로 작성됨.
- 뉴스 내용정리 시스템 개편요청
  1. 처음 수집된 뉴스는 먼저 올라마를 돌려서 같은 내용의 기사를 한번 그룹핑을 한다.
  2. 만들어진 그룹은 라이프보드에서 보여지는 하나의 뉴스가 되고 그 제목과 내용은 올라마가 잘 정리해서 실제 기사처럼 작성한다.(객관적인 정보만)
  3. 만들어진 기사는 아래에 관련 기사 링크로만 남겨두고 수집된 데이터는 삭제한다.
